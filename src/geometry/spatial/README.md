# three_dimension

## Overview

This module provides utilities for working with 3D geometry, including coordinate systems, length units, and 3D points.

## Components

| Component | Description |
|-----------|-------------|
| [coordinate_type.py](./coordinate_type.py) | Defines coordinate system types (Cartesian, Spherical, Cylindrical). |
| [cartesian_axis/](./cartesian_axis/README.md) | Utilities for Cartesian coordinate system axes, orientations, and handedness. |
| [point/](./point/README.md) | 3D point representation and operations. |
