# point

## Overview

This module provides utilities for working with 3D points, including single points, collections of points, and sequential point data.

## Components

| Component | Description |
|-----------|-------------|
| [point.py](./point.py) | Class representing a single 3D point with x, y, z coordinates. |
| [points.py](./points.py) | Class representing a collection of 3D points with batch operations. |
| [sequential.py](./sequential.py) | Class representing sequential 3D point data with frame IDs and interpolation capabilities. |
