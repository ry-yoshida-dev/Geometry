# point

## Overview

This module provides utilities for working with 2D points, including single points, collections of points, and sequential point data.

## Components

| Component | Description |
|-----------|-------------|
| [point.py](./point.py) | Class representing a single 2D point with x, y coordinates. |
| [points.py](./points.py) | Class representing a collection of 2D points with batch operations. |
| [sequential.py](./sequential.py) | Class representing sequential 2D point data with frame IDs and interpolation capabilities. |
