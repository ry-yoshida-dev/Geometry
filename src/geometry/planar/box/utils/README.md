# utils

## Overview

This module provides a set of utility functions for 2D bounding boxes.

## Components

| Component | Description |
|-----------|-------------|
| [box2d_calculator.py](./box2d_calculator.py) | Functions for calculations using 2D boxes such as IoU. |
| [box2d_converter.py](./box2d_converter.py) | Converts between 2D box coordinate formats. |
| [converters.py](./converters.py) | Functions for converting 2D bounding boxes to other formats. |
